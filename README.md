# Sales Data Analytics Dashboard

**Author:** Shashikant Yadav  
**Tech Used:** Python, Pandas, Matplotlib, Excel (openpyxl)

---

## 📌 Project Overview

End-to-end sales data analytics project — data cleaning, transformation, KPI analysis, and dashboard visualization across products, regions, and months.

---

## 📊 What This Project Does

- **Data Cleaning** — Handles missing values, duplicates, and data validation
- **Data Transformation** — Calculates Return Rate %, ROI %, MoM Growth
- **KPI Analysis** — Revenue, Net Revenue, Units Sold, Best Product/Region
- **Dashboard** — 6 charts: Revenue trends, Sales units, Product share, Region performance, Margin %, MoM Growth
- **Excel Report** — 5-sheet report: Raw Data, Monthly Trend, Product Analysis, Region Analysis, KPI Summary

## ▶️ How to Run

**1. Install dependencies:**
```bash
pip install pandas matplotlib openpyxl
```

**2. Run the script:**
```bash
python sales_analysis.py
```

**3. Outputs generated:**
- `Sales_Analytics_Report.xlsx` — 5-sheet Excel report
- `charts/sales_dashboard.png` — 6 visualizations
